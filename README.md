# Oddvice Web (PWA)

Next.js 16 + React 19 + Tailwind v4 **Progressive Web App**. Installable on
desktop, Android (Chrome), and iOS (Add to Home Screen).

## Requirements

- Node.js 18.18+ (tested on Node 24)

## Run

```bash
npm install
cp .env.example .env.local   # set NEXT_PUBLIC_API_URL
npm run dev                  # http://localhost:3000
```

`npm run build && npm start` for a production build.

## PWA features

- **Web app manifest** — generated by `src/app/manifest.ts`, served at
  `/manifest.webmanifest`.
- **Service worker** — `public/sw.js`: offline app-shell caching
  (network-first navigations, stale-while-revalidate assets) + Web Push
  handlers. Registered by `src/components/pwa.tsx`.
- **Offline fallback** — `src/app/offline/page.tsx`.
- **Install prompt** — `InstallPrompt` handles Chromium `beforeinstallprompt`
  and shows iOS A2HS instructions.
- **Icons** — regenerate with `node scripts/generate-icons.mjs`.

> Service workers only run over HTTPS (or `localhost`). For local push testing,
> use `next dev --experimental-https`.

## Configuration

| Var                   | Default                 | Description            |
| --------------------- | ----------------------- | ---------------------- |
| `NEXT_PUBLIC_API_URL` | `http://localhost:8080` | Base URL of the Go API |

The home page (`src/app/page.tsx`) shows a live API status badge using
`src/lib/api.ts`.

## Layout

```
src/app/            App Router pages, layout, manifest
src/components/     PWA registration, install prompt, API status
src/lib/api.ts      API client
public/             icons, service worker
```
